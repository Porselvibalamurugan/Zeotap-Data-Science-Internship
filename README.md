# Zeotap-Data-Science-Internship

# 🛒 eCommerce Transactions Data Science Assignment

This project focuses on analyzing an **eCommerce Transactions dataset** using **Data Science and Machine Learning techniques**. The objective is to perform **Exploratory Data Analysis (EDA)**, build a **Lookalike Recommendation Model**, and apply **Customer Segmentation using Clustering** to derive meaningful business insights.

## 📌 Tasks Covered

- **Exploratory Data Analysis (EDA)**
  - Data cleaning, visualization, and business insights generation.

- **Lookalike Model**
  - Recommending top 3 similar customers based on profile and transaction history.

- **Customer Segmentation**
  - Clustering customers using profile and transaction data with performance evaluation metrics.

## 🛠️ Technologies Used

- **Python**
- **Pandas, NumPy**
- **Matplotlib, Seaborn**
- **Scikit-learn**
- **Jupyter Notebook**


## 🎯 Goal

To extract actionable business insights, improve customer understanding, and apply machine learning techniques for recommendation and segmentation.
